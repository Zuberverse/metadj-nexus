## Mastering Workflow

1. **Source Files**: LANDR MP3 exports renamed to `NN - <Title> (v0) - Mastered.mp3` and stored in `~/Downloads/Majestic Ascent` as the canonical music files.
2. **Streaming Copies**: Upload those same 320 kbps MP3 files to App Storage (`audio-files/majestic-ascent/NN-track-title-mastered-v0.mp3`) for `/api/audio/...`.
3. **Metadata**: Track data updated in `src/data/tracks.json`; collection track count set to 39 in `src/data/collections.json`.
4. **Checks**: Run `npm run lint` and `npm run type-check` after metadata changes to ensure helpers compile cleanly.
